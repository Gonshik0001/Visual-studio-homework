# Мой Проект

## Измерение Покрытия

Проект использует Jest для измерения покрытия тестами. Вы можете измерить покрытие, запустив команду:

```bash
npm run coverage
Это создаст отчет о покрытии в папке coverage. Отчет включает информацию о покрытии функций, веток и строк.

Различия в Покрытии
Покрытие Функций (Functions): Измеряет, сколько функций в вашем коде было выполнено или протестировано.

Покрытие Веток (Branches): Измеряет, сколько ветвлений в вашем коде было выполнено или протестировано. Включает в себя условные операторы if/else и т.д.

Покрытие Строк (Lines): Измеряет, сколько строк вашего кода было выполнено или протестировано.

Важно добиться 100% покрытия для каждого из этих аспектов, чтобы убедиться в полной проверке вашего кода тестами.

Установка
Если вы еще не установили Jest, выполните следующую команду:
npm install --save-dev jest

Запуск Тестов
Вы можете запустить ваши тесты, используя команду:
npm test

Запуск с Измерением Покрытия
Чтобы запустить тесты с измерением покрытия, выполните:
npm run coverage

Обновление Тестов
Обновите ваши тесты, чтобы охватить все ветви кода и убедитесь, что отчет о покрытии отображает 100% для функций, веток и строк.

Отчет о Покрытии
Jest создаст отчет о покрытии в папке coverage. Откройте файл index.html в браузере, чтобы увидеть детальный отчет.
